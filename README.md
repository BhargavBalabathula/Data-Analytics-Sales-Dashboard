# Data-Analytics-Sales-Dashboard
📊 Data-Analytics-Sales-Dashboard
________________________________________
🚀 Project Overview
This project demonstrates an end-to-end data analysis workflow using Power BI, starting from raw, uncleaned sales data to building a fully interactive dashboard.
🔍 Key stages:
•	📥 Data Loading 
•	🧹 Data Cleaning (Power Query) 
•	🔗 Data Modelling 
•	🧮 DAX Calculations 
•	📊 Dashboard Creation 
📌 The final dashboard provides insights into sales performance, profitability, shipment trends, and regional distribution.
________________________________________
🎯 Objective
•	📊 Analyse raw sales data 
•	🧹 Make data analysis-ready 
•	🧮 Build KPIs using DAX 
•	📈 Create an interactive dashboard for decision-making 
________________________________________
🛠️ Tech Stack
Tool	Purpose
📊 Power BI Desktop	Dashboard development
🧹 Power Query	Data cleaning & transformation
🧮 DAX	KPI calculations
📂 Excel / CSV	Data source
💡 Entire project built inside Power BI (no external tools used)
________________________________________
🔄 Workflow
📥 Data Loading
•	Imported dataset into Power BI 
•	Verified structure & preview 
•	Selected relevant tables 
🧹 Data Cleaning
•	Removed duplicates 
•	Handled missing values 
•	Fixed data types 
•	Renamed columns 
•	Removed unnecessary fields 
🔄 Data Transformation
•	Formatted numerical columns 
•	Verified date columns 
•	Prepared data for grouping & bins 
🔗 Data Modelling
•	Created relationships between tables 
•	Defined fact & dimension tables 
•	Ensured correct filtering 
🧮 DAX KPIs
•	💰 Total Sales 
•	📈 Total Profit 
•	🚚 Total Shipments 
•	📦 Total Boxes 
•	📊 Profit % 
•	🔁 Previous Year (PY) Metrics
